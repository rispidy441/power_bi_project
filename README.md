# power_bi_project

This project includes an interactive powerbi sales dashboard built using:

1.Cleaned and transformed raw transactional data to ensure accuracy and consistency.
2.KPIs (Orders,Sales, Profit,ship days).
3.Charts (Column, Bar, Pie chart,Line chart and map).
4.Built a structured, management-ready dashboard to analyze monthly sales and profit trends, product categories & its sub- category, and customer performance(by region).

## Files Included

 1.power bi project (main dashboard) .
 2.power bi SuperStore_Sales_Dataset.csv (dataset used)

## Tools Used
   Power bi
